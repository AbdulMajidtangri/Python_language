content = """NUMPY LEARNING NOTES - CLEAR DESCRIPTION FILE

1. WHAT IS NUMPY?

NumPy stands for Numerical Python.

NumPy is a Python library used to perform fast numerical calculations using arrays. It is widely used in Data Science, Machine Learning, Artificial Intelligence, and mathematical operations.

Simple meaning:
NumPy helps us work with numbers, arrays, matrices, and mathematical calculations in a fast and professional way.


2. WHY NUMPY IS USED?

NumPy is used because normal Python lists are slower for large numerical data.

Main uses of NumPy:
- Fast mathematical calculations
- Working with arrays
- Working with matrices
- Data Science
- Machine Learning
- Statistical calculations
- Handling large numerical datasets


3. IMPORTING NUMPY

Before using NumPy, we import it.

Example:
import numpy as np

Meaning:
numpy = library name
np = short professional alias used for NumPy


4. CREATING A NUMPY ARRAY

A NumPy array is created using np.array().

Example:
arr = np.array([1, 2, 3, 4, 5])

Meaning:
np.array() converts a Python list into a NumPy array.


5. PYTHON LIST VS NUMPY ARRAY

Python list:
If we multiply a Python list by 2, it repeats the list.

Example:
list1 = [1, 2, 3]
print(list1 * 2)

Output:
[1, 2, 3, 1, 2, 3]

NumPy array:
If we multiply a NumPy array by 2, it multiplies each value.

Example:
arr = np.array([1, 2, 3])
print(arr * 2)

Output:
[2 4 6]

Difference:
Python list repeats values.
NumPy array performs mathematical operation on each value.


6. 1D ARRAY

A 1D array is a simple single-row array.

Example:
arr = np.array([1, 2, 3, 4, 5])

Output:
[1 2 3 4 5]


7. 2D ARRAY

A 2D array is like a table with rows and columns.

Example:
arr = np.array([
    [1, 2],
    [3, 4]
])

Output:
[[1 2]
 [3 4]]


8. LOOPING THROUGH NUMPY ARRAY

Looping means accessing each value one by one.

Looping through 1D array:

arr = np.array([1, 2, 3, 4, 5])

for value in arr:
    print(value)

Looping through 2D array:

arr = np.array([
    [1, 2],
    [3, 4]
])

for row in arr:
    for value in row:
        print(value)


9. CHECKING ARRAY TYPE

type() is used to check the object type.

Example:
print(type(arr))

Output:
<class 'numpy.ndarray'>

Meaning:
ndarray means N-dimensional array.


10. CHECKING ARRAY DIMENSION

ndim is used to check the dimension of an array.

Example:
print(arr.ndim)

Meaning:
1D array returns 1.
2D array returns 2.
3D array returns 3.


11. CHECKING ARRAY SHAPE

shape is used to check the structure of an array.

Example:
print(arr.shape)

For 1D array:
(5,)

Meaning:
The array has 5 elements.

For 2D array:
(2, 3)

Meaning:
The array has 2 rows and 3 columns.


12. CHECKING ARRAY SIZE

size is used to check the total number of elements in an array.

Example:
print(arr.size)

For this array:
arr = np.array([
    [1, 2],
    [3, 4]
])

Size will be:
4

Meaning:
There are 4 total elements in the array.


13. CHECKING DATA TYPE

dtype is used to check the data type of array values.

Example:
print(arr.dtype)

Possible outputs:
int32
int64
float64
<U7

Important point:
If a NumPy array contains numbers and strings together, NumPy may convert all values into string.


14. ARRAY INDEXING

Indexing means accessing a value using its position number.

Example:
arr = np.array([10, 20, 30, 40])

print(arr[0])
print(arr[-1])

Meaning:
arr[0] gives the first value.
arr[-1] gives the last value.

Important:
Index starts from 0.


15. ARRAY SLICING

Slicing means getting a specific part of an array.

Example:
arr = np.array([1, 2, 3, 4, 5, 6])

print(arr[1:4])

Output:
[2 3 4]

Syntax:
array[start:end:step]

Important:
End index is not included.


16. ZEROS ARRAY

np.zeros() creates an array filled with zeros.

Example:
arr = np.zeros(5)

Output:
[0. 0. 0. 0. 0.]

2D example:
arr = np.zeros((3, 3))

Meaning:
This creates 3 rows and 3 columns filled with zeros.


17. ONES ARRAY

np.ones() creates an array filled with ones.

Example:
arr = np.ones(5)

Output:
[1. 1. 1. 1. 1.]

2D example:
arr = np.ones((2, 3))

Meaning:
This creates 2 rows and 3 columns filled with ones.


18. ARANGE FUNCTION

np.arange() works like Python range(), but it creates a NumPy array.

Example:
arr = np.arange(1, 6)

Output:
[1 2 3 4 5]

With step:
arr = np.arange(1, 10, 2)

Output:
[1 3 5 7 9]

Meaning:
Start from 1, stop before 10, and increase by 2.


19. LINSPACE FUNCTION

np.linspace() creates a fixed number of equally spaced values between start and end.

Example:
arr = np.linspace(1, 10, 5)

Meaning:
Start from 1, end at 10, and create 5 values with equal distance.

Important:
In linspace(), the end value is included by default.


20. ARITHMETIC OPERATIONS

NumPy allows direct mathematical operations on arrays.

Example:
arr = np.array([1, 2, 3, 4])

print(arr + 5)
print(arr - 1)
print(arr * 2)
print(arr / 2)

Meaning:
The operation is applied to every element of the array.


21. ARITHMETIC OPERATIONS BETWEEN TWO ARRAYS

Example:
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])

print(arr1 + arr2)
print(arr1 * arr2)

Meaning:
Values at the same positions are calculated together.

Important:
Both arrays should have compatible shapes.


22. STATISTICAL OPERATIONS

NumPy provides many statistical functions.

Example:
arr = np.array([10, 20, 30, 40])

np.sum(arr) gives total.
np.max(arr) gives highest value.
np.min(arr) gives lowest value.
np.mean(arr) gives average.
np.median(arr) gives middle value.
np.std(arr) gives standard deviation.


23. MODULUS OPERATION

np.mod() gives the remainder.

Example:
arr = np.array([1, 2, 3, 4])

print(np.mod(arr, 2))

Output:
[1 0 1 0]

Meaning:
Each value is divided by 2 and the remainder is returned.


24. RESHAPE FUNCTION

reshape() is used to change the shape of an array.

Example:
arr = np.array([1, 2, 3, 4])

new_arr = arr.reshape(2, 2)

Output:
[[1 2]
 [3 4]]

Important:
Total number of elements must remain the same.

Examples:
4 elements can become 2 x 2.
6 elements can become 2 x 3.
8 elements can become 4 x 2.


25. SORTING ARRAY

np.sort() is used to sort an array.

Example:
arr = np.array([4, 1, 3, 2])

print(np.sort(arr))

Output:
[1 2 3 4]

Descending order:
print(np.sort(arr)[::-1])

Meaning:
[::-1] reverses the sorted array.


26. COPY IN NUMPY

copy() creates an independent array.

Example:
copy_arr = arr.copy()

Meaning:
Changing the original array does not affect the copy.
Changing the copy does not affect the original array.


27. VIEW IN NUMPY

view() creates a dependent array.

Example:
view_arr = arr.view()

Meaning:
Changing the original array affects the view.
Changing the view can also affect the original array.


28. DIFFERENCE BETWEEN COPY AND VIEW

copy:
Independent array.
Changes do not affect original.

view:
Dependent array.
Changes affect original data.


29. JOINING ARRAYS

Joining means combining two or more arrays into one array.

Example:
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])

result = np.concatenate((arr1, arr2))

Output:
[1 2 3 4 5 6]

Meaning:
concatenate() joins arrays in the same direction.


30. STACKING ARRAYS

Stacking means arranging arrays row-wise or column-wise by creating a new axis.

Example:
arr1 = np.array([1, 2, 3])
arr2 = np.array([4, 5, 6])

np.stack((arr1, arr2), axis=0)

Output:
[[1 2 3]
 [4 5 6]]

Meaning:
axis=0 stacks arrays row-wise.


31. STACKING WITH AXIS 1

Example:
np.stack((arr1, arr2), axis=1)

Output:
[[1 4]
 [2 5]
 [3 6]]

Meaning:
axis=1 stacks arrays column-wise.


32. HSTACK FUNCTION

hstack() means horizontal stacking.

Example:
np.hstack((arr1, arr2))

Output:
[1 2 3 4 5 6]

Meaning:
It joins arrays side by side.


33. VSTACK FUNCTION

vstack() means vertical stacking.

Example:
np.vstack((arr1, arr2))

Output:
[[1 2 3]
 [4 5 6]]

Meaning:
It joins arrays top to bottom.


34. AXIS IN NUMPY

Axis tells NumPy the direction of operation.

axis=0:
Column-wise operation.

axis=1:
Row-wise operation.

Example:
arr = np.array([
    [1, 2],
    [3, 4]
])

np.sum(arr, axis=0)

Output:
[4 6]

Meaning:
1 + 3 = 4
2 + 4 = 6

np.sum(arr, axis=1)

Output:
[3 7]

Meaning:
1 + 2 = 3
3 + 4 = 7


35. CONCATENATE 2D ARRAYS

Example:
arr1 = np.array([
    [1, 2],
    [3, 4]
])

arr2 = np.array([
    [5, 6],
    [7, 8]
])

np.concatenate((arr1, arr2), axis=0)

Output:
[[1 2]
 [3 4]
 [5 6]
 [7 8]]

Meaning:
axis=0 joins rows.


36. CONCATENATE 2D ARRAYS COLUMN-WISE

Example:
np.concatenate((arr1, arr2), axis=1)

Output:
[[1 2 5 6]
 [3 4 7 8]]

Meaning:
axis=1 joins columns.
